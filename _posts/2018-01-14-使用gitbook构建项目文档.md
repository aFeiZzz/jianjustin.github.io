# gitbook使用入门
## 基本使用

* 安装gitbook(需要先安装node获取npm)
    npm install gitbook-cli -g
* 创建book*gitbook init*
* 构建并运行book*gitbook serve*
or
* 构建编译gitbook*gitbook build*
